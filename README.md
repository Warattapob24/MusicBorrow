# ระบบยืมคืนเครื่องดนตรี (Supabase + JavaScript)

## วิธีใช้งาน
1. เปิดไฟล์ `index.html` เพื่อเข้าสู่ระบบ
2. สมัครสมาชิกได้ที่ `register.html`
3. เข้าสู่ระบบสำเร็จจะไปที่ `dashboard.html`
4. ฟีเจอร์ admin ใช้งานได้เมื่อเข้าสู่ระบบด้วยบัญชี admin

## โครงสร้างไฟล์
- index.html, login.js — หน้าเข้าสู่ระบบ
- register.html, register.js — สมัครสมาชิก
- dashboard.html, dashboard.js — ฟีเจอร์หลัก
- logout.js — ออกจากระบบ

## การตั้งค่า Supabase
- แก้ไข URL และ anon key ในไฟล์ JS ให้ตรงกับโปรเจกต์ของคุณ
- ตรวจสอบ RLS และ Policy ใน Supabase

## หมายเหตุ
- ใช้งานได้ทั้งบนคอมและมือถือ
- ต้องเชื่อมต่ออินเทอร์เน็ต
